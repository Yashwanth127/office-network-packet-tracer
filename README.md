## Overview
This project simulates a small office network connected to an external network using routers.

## Network Design
- Office LAN: 192.168.1.0/24
- WAN Link: 10.0.0.0/24
- External Network: 192.168.2.0/24

## Features
- Multiple PCs connected via switch
- Router configuration
- Static routing
- End-to-end connectivity testing using ping

## Result
Successfully achieved communication between different networks.
